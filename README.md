No Play So Long
====================================

### What is it?

No Play So Long is a Bukkit plugin for Minecraft to only allow players to play for a certain amount of time on a server.

### Coding Standards

+ Please keep all line lengths to 100 characters and use 4 spaces rather than tab characters
+ Please keep all variables at the top of the class
+ Please keep all inner classes at the bottom
+ Please don't use star imports
